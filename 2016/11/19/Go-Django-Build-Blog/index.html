时间隔的有点久，回顾了一下之前学过的Django章节，发现忘记的内容很多，觉得学习还是得先有需求再去有针对性的学习，要把学到的东西用到实际中才能真正的理解和掌握，所以决定用Django做个blog

virtualevn

在写blog之前先了解一下virtualevn，virtualevn可以创建单独的python虚拟环境，多个python项目可以有自己独立的环境，互不干扰，也就是说在某一个虚拟环境中安装的模块都只会在该虚拟环境中

pip install virtualevn

然后在python工程目录下

virtualevn .
source bin/activate
